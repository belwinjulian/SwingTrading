| schema → consumer | RankingSnapshotSchema extension; downstream report renderer trusts pandera-validated dtypes |
| read_universe_latest → caller | Returns list[str] of tickers from the newest universe snapshot; lexicographic sort over ISO-date filenames is safe because `_universe_dir()` is settings-controlled (not user input) |
